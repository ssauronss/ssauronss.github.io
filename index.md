   Click on the file that corresponds to your operating system. This might be a `.crx` file for Chrome users.

## 🔍 How to Use Omnibar

- **Open the Command Palette**: Press `Ctrl + Shift + P` (or `Cmd + Shift + P` on macOS) to open Omnibar.
- **Switch Tabs**: Start typing the name of the tab you want to switch to. Omnibar will display relevant options for you to choose from.
- **Search History**: Type keywords related to your past browsing. Omnibar will help you find your history quickly.
- **Web Navigation**: Type a website name, and use the commands to navigate directly.

## ⚙️ Features

- **Fuzzy Search**: Omnibar can understand partial names, making it easier to find what you need quickly.
- **Tab Switching**: Save time by quickly navigating between your open tabs.
- **Web Search**: Easily search the web without waiting for separate searches.
